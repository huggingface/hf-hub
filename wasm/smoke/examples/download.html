hf-hub — wasm download example

Build the WASM package first: ./build_wasm.sh, then serve this directory with a COOP/COEP-enabled static server (required for the threaded wasm in hf-xet) and open http://localhost:8080/examples/download.html.

The page calls HFRepository::download_file_stream via wasm-bindgen — the same builder used on native — and either reports the byte total or collects the bytes for a Save button.
